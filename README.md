Yummy — Meal Finder App
A responsive web app to explore meals and recipes from around the world using TheMealDB API.

[Live Demo](https://omarkhattab74.github.io/yummy/)

🚀 Features
🔍 Search meals by name or first letter
🗂️ Browse by category , area and ingredients
📋 View full recipe details & ingredients
🎬 Watch YouTube tutorials per meal
📱 Fully responsive design

🛠️ Tech Stack
HTML , CSS, JavaScript , Bootstrap 5, Font Awesome, TheMealDB API
